# 🍎 Apple Design OS for Agents

## 1. System Overview

Apple Design OS is a **constraint-driven design intelligence system** that enables AI agents to:

- Generate product UI/UX
- Evaluate interface quality
- Enforce cognitive design constraints
- Improve designs iteratively

It transforms Apple Design Principles into a **runtime decision system**, not a guideline.

---

## 2. System Architecture

### Layer 1: Design Prior (What is good design)
- Clarity
- Deference
- Depth
- UX heuristics

### Layer 2: Constraint Engine (What is allowed)
- No UI > content dominance
- Motion must explain structure
- No ambiguous affordances
- No inconsistent interaction models

### Layer 3: Evaluation Engine (What is better)
- Multi-dimensional scoring system
- Anti-pattern detection
- Cognitive load estimation

### Layer 4: Generation Engine (What to produce)
- UI structure
- Interaction flows
- Layout hierarchy
- Motion model

---

## 3. Core Principle

> Apple Design OS does NOT design UI.
> It constrains and evaluates the space in which UI can exist.

---

## 4. Agent Workflows

### A. Generate Mode
Input:
- Product goal
- User intent

Output:
- UI specification constrained by Apple rules

---

### B. Evaluate Mode
Input:
- UI mock / description

Output:
- Score breakdown
- Violations
- Fix recommendations

---

### C. Improve Mode
Input:
- Existing UI design

Output:
- Optimized version under constraints

---

## 5. Design Constraints

### Clarity Constraint
- No guessing required
- Every screen must declare:
  - location
  - action
  - next step

### Deference Constraint
- Content > UI
- UI must not compete with content

### Depth Constraint
- Motion = structure explanation
- Layers = hierarchy mapping

---

## 6. Output Standard

All agent outputs must include:

1. Structure explanation
2. Constraint compliance check
3. Scoring evaluation
4. Improvement suggestions

---

## 7. System Goal

To create an AI-native design system where:

> good design is not subjective aesthetic judgment,
> but a constrained optimization problem over cognition, clarity, and interaction cost.
