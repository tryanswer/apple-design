# 🍎 Apple Design Agent Skill

This repository encodes Apple’s design philosophy (Clarity, Deference, Depth) into an executable Agent Skill for UI/UX generation, evaluation, and design reasoning.

## Core Principles

### 1. Clarity
Users should instantly understand:
- Where they are
- What they can do
- What happens next

### 2. Deference
UI should step back and let content lead.

### 3. Depth
Motion and layering should explain system structure, not decorate it.

## UX Heuristics

- Reduce cognitive load
- Make interfaces invisible in flow
- Use physical metaphors consistently
- Maintain system-wide consistency
- Deliver surprise via effort reduction

## Skill Files

- `skill.md`: Full executable Apple Design Agent Skill

## Usage
This repo is intended to be used as a **design reasoning module for AI agents**, not just a UI guideline.

It can be embedded into:
- Product design agents
- UI generation systems
- Design review automation
- Agent interface frameworks
